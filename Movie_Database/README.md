# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh


Features

Search for movies by name.
View a list of movies from multiple genres.
Display detailed information about a selected movie.
Get movie suggestions if no results are found.
Refresh the movie list or search results.
View a loading spinner while data is being fetched.

Technologies Used

React - JavaScript library for building user interfaces.
Axios - Promise-based HTTP client for making API requests.
OMDb API - API to fetch movie data.
Tailwind CSS - Utility-first CSS framework for styling.

How to Use

Search for Movies: Use the search bar at the top to enter the movie name. The app will show matching results from the OMDb API.
View Movie Details: Click on any movie to view detailed information, including the plot, cast, and ratings.
Watch Trailers: You can view trailers by clicking the "Watch Trailer" link for each movie (it redirects to YouTube).
Movie Suggestions: If no movies match your search, the app will show a list of suggested movies to try.
